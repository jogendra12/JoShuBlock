Jo Shu Block is a collection of Gutenberg blocks that enhance the WordPress block editor with unique and customizable blocks.

== Description ==

Jo Shu Block provides a set of creative and customizable blocks for the Gutenberg editor, allowing users to design engaging layouts easily. Whether you're a developer or a designer, these blocks help extend WordPress's default block editor functionality.

Features:

Custom Gutenberg blocks for enhanced content creation.

Lightweight and optimized for performance.

Seamless integration with the WordPress editor.

Regular updates and compatibility with the latest WordPress versions.

== Installation ==

Upload the jo-shu-block folder to the /wp-content/plugins/ directory.

Activate the plugin through the 'Plugins' menu in WordPress.

Open the WordPress block editor and start using the new blocks.

== Frequently Asked Questions ==

= How do I use Jo Shu Block? =
After activation, open the block editor and search for "Jo Shu Block" in the blocks list. Drag and drop the blocks into your content.

= Is this plugin compatible with all themes? =
Yes, Jo Shu Block works with all modern WordPress themes that support the block editor.

= Does this plugin slow down my website? =
No, Jo Shu Block is optimized for performance and follows best coding practices.
